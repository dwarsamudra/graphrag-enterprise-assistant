# GraphRAG Enterprise Assistant 🚀

A practical enterprise-style GraphRAG project built using Neo4j, Ollama, Llama3, embeddings, and Streamlit.

## Features
- Hybrid retrieval using graph + vector search
- Policy chunking and semantic search
- Neo4j graph relationships
- Chat history
- Confidence score
- Downloadable answer report
- Streamlit UI for enterprise policy Q&A

## Tech Stack
- Python
- Streamlit
- Neo4j
- Ollama
- Llama3
- Nomic Embeddings
- NumPy
